# 🛠️ محلل الكود البرمجي

موقع ويب لتحليل جودة الكود البرمجي بعدة لغات برمجة.

## 🌟 الميزات

- ✅ تحليل كود **Python**
- ✅ تحليل كود **Lua/Luau** (لـ Roblox)
- ✅ تحليل كود **C, C++, C#**
- ✅ كشف المشاكل الأمنية
- ✅ تحليل الأداء
- ✅ نصائح لتحسين الجودة

## 🚀 الاستخدام

1. اختر لغة البرمجة
2. الصق الكود في المنطقة المخصصة
3. اضغط "تحليل الكود"
4. احصل على تقرير مفصل
